# Asset Policy

This repository does not ship third-party sports logos or card photography.

## Included assets
- `public/placeholders/card.svg`
- `public/placeholders/logo.svg`
- app-owned project icons (`public/favicon.svg`, `public/vite.svg`)

## Notes
- Historical third-party card/logo assets and download scripts were removed for open-source safety.
- If you add real logos or card images, only use assets you own or are licensed to redistribute.
- Keep attribution and license terms with any newly added third-party assets.
